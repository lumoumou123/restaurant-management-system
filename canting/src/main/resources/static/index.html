餐厅管理系统 API

这是餐厅管理系统的后端API服务。以下是可用的API端点：

健康检查

  • GET /api/health - 检查API服务状态

数据资源

  • GET /api/list - 获取列表数据

餐厅管理

  • GET /api/canteen/list - 获取餐厅列表
  • POST /api/canteen - 添加新餐厅
  • PUT /api/canteen - 更新餐厅信息
  • GET /api/canteen/owner/canteens - 获取当前用户的餐厅

统计数据

  • GET /statistics/canting - 获取餐厅统计数据
  • GET /statistics/summary - 获取统计数据汇总
  • GET /statistics/owner - 获取业主所有餐厅的统计数据

菜单管理

  • GET /api/menu-items/canting/{canteenId} - 获取餐厅的菜单
  • POST /api/menu-items - 添加菜单项
  • PUT /api/menu-items - 更新菜单项

评论管理

  • GET /api/comments/canteen/{canteenId} - 获取餐厅的评论
  • POST /api/comments - 添加评论
  • DELETE /api/comments/{id} - 删除评论

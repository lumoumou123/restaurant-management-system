Restaurant Management System API

This is the backend API service for the Restaurant Management System. Below are the available API endpoints:

Health Check

  • GET /api/health - Check API service status

Data Resources

  • GET /api/list - Get list data

Restaurant Management

  • GET /api/canteen/list - Get restaurant list
  • POST /api/canteen - Add new restaurant
  • PUT /api/canteen - Update restaurant information
  • GET /api/canteen/owner/canteens - Get current user's restaurants

Statistics

  • GET /statistics/canting - Get restaurant statistics
  • GET /statistics/summary - Get statistics summary
  • GET /statistics/owner - Get statistics for all restaurants owned by the owner

Menu Management

  • GET /api/menu-items/canting/{canteenId} - Get restaurant menu
  • POST /api/menu-items - Add menu item
  • PUT /api/menu-items - Update menu item

Comment Management

  • GET /api/comments/canteen/{canteenId} - Get restaurant comments
  • POST /api/comments - Add comment
